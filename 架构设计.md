# 架构设计

当前版已迁移到 [`docs/architecture（架构设计）.md`](./docs/architecture（架构设计）.md)。

保留这个根目录文件是为了兼容旧链接。请以新文档为准，它已经按当前代码重新核对：

- 项目默认主线是 `main`（分布式 Agent 大脑）；本分支 `next/server-brain` 是 Server 大脑并行架构线。
- Server 负责 VLM 决策、轨迹缓存、辅助系统、调度、报告。
- Agent 负责发现设备、镜像、执行 Server 下发的 driver command。
- iOS 推荐 stable WDA 生命周期；Android / HarmonyOS 推荐黑屏待机、Run 前唤醒。

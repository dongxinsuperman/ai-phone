# 架构设计

当前版已迁移到 [`docs/architecture（架构设计）.md`](./docs/architecture（架构设计）.md)。

保留这个根目录文件是为了兼容旧链接。请以新文档为准，它已经按当前代码重新核对：

- 默认主线是 `main`（分布式 Agent 大脑）；`next/server-brain`（Server 大脑）为并行架构线。
- 分布式 Agent 大脑：Agent 本地跑 VLM 决策 + 执行安全层 + 缓存回放 / 第一手归档；Server 集中下发配置 / 密钥、命中下发缓存 + 薄存储、调度、报告。
- 两条线数据库 schema 兼容、二选一部署；分支对照见 README「分支说明」。
- iOS 推荐 stable WDA 生命周期；Android / HarmonyOS 推荐黑屏待机、Run 前唤醒。
